## 🏡 ApnaStay Anywhere

ApnaStay Anywhere is a modern, full-stack rental property management platform inspired by Airbnb. It enables hosts to list and manage properties, set pricing, control availability, and handle bookings, while guests can explore stays, check real-time availability, book securely, and leave reviews.

The platform is built using Next.js with TypeScript for a scalable frontend, styled with Tailwind CSS and shadcn/ui for a clean, responsive user experience. Supabase (PostgreSQL) is used for authentication, database management, and real-time data handling. Stripe (test mode) is integrated to support secure payment workflows.

Designed with production-ready architecture and deployed on AWS, ApnaStay Anywhere focuses on performance, security, and scalability, providing a seamless and reliable experience for both hosts and travelers.
